# CrowdNet


## 数据集获取
 - 数据集下载链接：https://aistudio.baidu.com/aistudio/datasetdetail/1917
 - 把`train.json`文件存放在当前目录
 - 把`test_new.zip`解压到当前目录
 - 把`train_new.zip`解压到当前目录
 
 
## 训练模型
 - 执行`python3 train.py`开始训练模型
 

## 预测
 - 执行`python3 infer.py`预测图片，获取图像中人的数量
